## Conseils d'usage
- Pour un PDF : ouvrir le `.md` dans un éditeur Markdown (Typora, VS Code +
  export) ou coller dans Google Docs.
- Pour un deck : une section `## Slide N` = une diapo.
- **Honnêteté** : utiliser les **stats réseau réelles** (page d'accueil / admin),
  ne pas inventer de chiffres ; ne pas promettre de gains passifs 24/7 (cf.
  contraintes iOS).
